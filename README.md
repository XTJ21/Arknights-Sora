# 使用说明 Instruction
双击运行main.py  
进入需要连续行动的关卡，在开始行动界面点击Arknights-Sora"start"按钮  

注： 
1，仅适用于支持adb调试的安卓模拟器 
2，adb连接手机需要开启开发者模式
